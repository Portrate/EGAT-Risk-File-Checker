ใช้ PyMuPDF ดึงข้อความโดยตรง เปรียบเสมือนการคัดลอกข้อความจากหน้าจอ

⚡

PDF จากการสแกน

เอกสารที่ได้จากเครื่องสแกน หรือเป็นการนำภาพถ่ายมาใส่ ไม่มีข้อความฝังอยู่ มีเพียงรูปภาพของตัวหนังสือ

วิธีการ

การแบ่งข้อความ (Chunking)

AI แต่ละโมเดลมีขีดจำกัดปริมาณข้อความที่รับได้ต่อครั้ง หากเอกสารมีขนาดใหญ่ ต้องแบ่งเป็นส่วนย่อยก่อน

ขนาดข้อความ (Chunk Size)

24,000

ตัวอักษร (ประมาณ 6,000–8,000 คำ)

ข้อความทับซ้อน (Overlap)

2,000

ตัวอักษร (ป้องกันเนื้อหาตรงรอยต่อสูญหาย)

หลักการทำงาน

1

แบ่งข้อความตามตัวคั่นหน้า [หน้า N] เพื่อไม่ตัดข้ามหน้า

2

รวมหน้าจนครบ 24,000 ตัวอักษร แล้วเริ่มชิ้นใหม่

3

ทำซ้ำจนครบทุกหน้าของเอกสาร

4

ส่งชิ้นข้อความทีละ Chunk ให้ AI วิเคราะห์

⚡ เทคนิคเพิ่มความเร็ว: หาก AI พบว่ารายการนั้น "ผ่าน" ใน Chunk ใดแล้ว จะหยุดทันทีโดยไม่ตรวจ Chunk ที่เหลือ

การวิเคราะห์ด้วย AI

ระบบสร้างคำสั่ง (Prompt) ที่มีโครงสร้างชัดเจน ส่งให้ AI ตัดสินทีละรายการ โดยกำหนดเกณฑ์อย่างเข้มงวด

สิ่งที่ส่งให้ AI ทุกครั้ง

บทบาท "ผู้ตรวจสอบเอกสารที่เข้มงวด"

ข้อกำหนดที่ต้องตรวจ (จาก Checklist)

ข้อความจากเอกสาร (แต่ละชิ้น)

เกณฑ์ตัดสิน 9 ข้อ ที่ต้องปฏิบัติอย่างเคร่งครัด

รูปแบบคำตอบ JSON ที่กำหนดไว้ชัดเจน

AI ต้องตอบกลับเป็น JSON

{
  "status": "pass" หรือ "fail"
  "reasoning": "เหตุผลเป็นภาษาไทย"
  "evidence": "ข้อความจากเอกสาร"
  "found_in_document": true/false
}

กลไกป้องกัน: หาก AI ตอบไม่ตรงรูปแบบ ระบบมี regex fallback แยกแต่ละฟิลด์ออกมาได้ แม้ JSON จะถูกตัดกลางคัน

กลไกรับมือข้อผิดพลาดอัตโนมัติ

Exponential Backoff Retry

เมื่อ Gemini หรือ OpenAI ตอบกลับด้วยรหัส 429 (เกินโควตา) หรือ 503 (เซิร์ฟเวอร์เต็ม) ระบบจะรอแล้วลองใหม่โดยเพิ่มเวลารอเป็นทวีคูณ สูงสุด 5 ครั้ง

1 วินาที → 2 วินาที → 4 วินาที → 8 วินาที → 16 วินาที

JSON Fallback Parser

หาก AI ส่ง JSON ที่ไม่สมบูรณ์ (เช่น ถูกตัดกลางคัน) ระบบใช้ Regex แยกค่าแต่ละฟิลด์ออกมาทีละตัว เพื่อให้ได้ผลลัพธ์ออกมา

Thai Language Retry

ตรวจสอบว่าคำตอบที่ AI ตอบเป็นภาษาไทยหรือไม่ (ต้องมีตัวอักษรไทยอย่างน้อย 15%) หากเป็นภาษาอังกฤษ ระบบจะส่งคำสั่งใหม่ด้วยข้อความเน้นย้ำกว่าเดิม

Sanitize Reasoning

บางครั้ง AI อาจสร้างข้อความซ้ำวนไปมา ระบบจะตรวจจับและตัดส่วนที่ซ้ำออกอัตโนมัติ ทั้งระดับตัวอักษรและระดับคำ

Pydantic Validation

ตรวจสอบโครงสร้าง Checklist ก่อนประมวลผล หากไม่ถูกต้องจะแจ้งกลับทันทีพร้อมรายละเอียดข้อผิดพลาด
