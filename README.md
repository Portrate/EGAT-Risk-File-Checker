# Risk File Checker — ระบบตรวจสอบเอกสาร PDF

ระบบตรวจสอบเอกสาร PDF ว่ามีเนื้อหาครบถ้วนตามข้อกำหนดที่ผู้ใช้กำหนดเอง
โดยใช้ Local LLM (Gemma 4) เพื่อรักษาความลับของข้อมูล

---

## ความต้องการของระบบ

- Windows 10 / 11
- RAM อย่างน้อย **8 GB** (แนะนำ 16 GB ขึ้นไป)
- พื้นที่ว่างอย่างน้อย **10 GB** (สำหรับโมเดล AI)

---

## ขั้นตอนที่ 1 — ติดตั้ง Python

1. เปิดเบราว์เซอร์ไปที่ https://www.python.org/downloads/
2. คลิก **"Download Python 3.x.x"** (เวอร์ชันล่าสุด)
3. เปิดไฟล์ติดตั้ง
4. ติ๊กถูกที่ช่อง **"Add Python to PATH"** ก่อนกด Install
5. คลิก **"Install Now"** และรอจนเสร็จ
6. ตรวจสอบการติดตั้งโดยเปิด Command Prompt แล้วพิมพ์:
   ```
   python --version
   ```
   ถ้าขึ้น `Python 3.x.x` แสดงว่าติดตั้งสำเร็จ

---

## ขั้นตอนที่ 2 — ติดตั้ง Ollama

Ollama คือโปรแกรมที่ใช้รัน AI บนเครื่อง

1. เปิดเบราว์เซอร์ไปที่ https://ollama.com
2. คลิก **"Download"** และเลือก **Windows** (หากใช้ระบบปฏิบัติการ Windows)
3. เปิดไฟล์ตืดตั้ง `OllamaSetup.exe` แล้วกด Install
4. หลังติดตั้งเสร็จ Ollama จะรันอยู่เบื้องหลังอัตโนมัติ (มีไอคอนแสดงใน System Tray)
5. ตรวจสอบการติดตั้งโดยเปิด Command Prompt แล้วพิมพ์:
   ```
   ollama --version
   ```
   ถ้าขึ้นเลขเวอร์ชันแสดงว่าติดตั้งสำเร็จ

---

## ขั้นตอนที่ 3 — ดาวน์โหลดโปรแกรม

**ดาวน์โหลด ZIP (แนะนำ):**
1. ไปที่หน้า GitHub ของโปรเจกต์
2. คลิกปุ่มสีเขียว **Code** → **Download ZIP**
3. แตกไฟล์ ZIP ไปไว้ในโฟลเดอร์ที่ต้องการ

---

## ขั้นตอนที่ 4 — เปิดโปรแกรม

1. เปิดโฟลเดอร์โปรเจกต์
2. ดับเบิลคลิกที่ไฟล์ **`start.bat`**
3. หน้าต่าง Command Prompt จะเปิดขึ้นและทำงานต่อไปนี้โดยอัตโนมัติ:

   ```
   [1/3] Creating virtual environment...   ← สร้างสภาพแวดล้อม Python
   [2/3] Installing dependencies...        ← ติดตั้ง Library ที่จำเป็น
   [3/3] Checking Ollama model...          ← ตรวจสอบ/ดาวน์โหลดโมเดล AI
   ```

   > **หมายเหตุ:** การเปิดครั้งแรกอาจใช้เวลานาน (5–15 นาที) ในการดาวน์โหลดโมเดล Gemma 4 (ขนาดประมาณ 9 GB)
   > ในการเปิดครั้งต่อไปจะเปิดเร็ว

4. เมื่อพร้อมใช้งาน เบราว์เซอร์จะเปิดขึ้นอัตโนมัติที่ http://localhost:8000

---

## ขั้นตอนที่ 5 — ใช้งานระบบ

1. **กรอก Checklist** — เพิ่มหัวข้อใหญ่และหัวข้อย่อยที่ต้องการตรวจสอบ
2. **อัปโหลด PDF** — ลากไฟล์มาวางหรือคลิกเลือกไฟล์
3. **กด "ตรวจสอบ"** — ระบบจะวิเคราะห์เอกสารและแสดงผล
4. **ดูผลลัพธ์** — แต่ละหัวข้อจะแสดงสถานะ ผ่าน / ไม่ผ่าน พร้อมหลักฐานจากเอกสาร

---

## การปิดโปรแกรม

กด `Ctrl + C` ในหน้าต่าง Command Prompt ที่รัน start.bat

---

## แก้ไขปัญหาเบื้องต้น

| ปัญหา | วิธีแก้ |
|---|---|
| `Python not found` | ติดตั้ง Python ใหม่ โดยติ๊ก **"Add Python to PATH"** |
| `Ollama not found` | ติดตั้ง Ollama จาก https://ollama.com แล้วรีสตาร์ท Command Prompt |
| เชื่อมต่อ Ollama ไม่ได้ | คลิกขวา Taskbar → ตรวจสอบว่ามีไอคอน Ollama อยู่ใน System Tray ถ้าไม่มีให้เปิด Ollama ก่อน |
| หน้าเว็บไม่เปิด | เปิดเบราว์เซอร์แล้วพิมพ์ `http://localhost:8000` ด้วยตนเอง |
| โมเดลดาวน์โหลดช้า | ขึ้นอยู่กับความเร็วอินเทอร์เน็ต โมเดลมีขนาดประมาณ 9 GB |
| ผลลัพธ์ช้ามาก | Gemma 4 ต้องใช้ RAM อย่างน้อย 8 GB ถ้ามี GPU จะเร็วขึ้น |
| `Port 8000 already in use` | มีโปรแกรมอื่นใช้ port อยู่ — รีสตาร์ทเครื่องหรือปิดโปรแกรมอื่นก่อน |
